# Executive Mom
---
A todo list app for moms. Also promotes "be kind to yourself" philosophy by click to display positive message.

Creator:
Mariela Chavez(@mau211)

## Technologies Used

### Back-end
1. PostgreSQL
2. Docker
3. Spring Boot

### Front-end
1. Bootstrap
2. Axios
3. Formik
4. Moment
5. React
6. Amazon Web Services
